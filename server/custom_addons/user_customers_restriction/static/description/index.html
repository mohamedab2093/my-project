Customer Restriction on Users

User Can See and Manage Allowed Products Only



  • This Module adds restriction on users for accessing customer for any kind of operation.
  • Admin can edit the user and add allowed customer to a specific user.
  • User can not see the customer if not allowed by the admin.

  • Restriction also applies to sales order, purchase order, stock transfer etc.
  • User can only see and operate on Allowed customers.
  • Note: Restriction is Not Applied On Adminstrator.
