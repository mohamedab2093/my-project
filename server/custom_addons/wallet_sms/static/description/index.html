Two layer Authentication

Odoo SMS OTP Authentication: An impressive application which adds extra layer of security and provides an option to log-in/sign-up via OTP.

Information

User Guide
https://webkul.com/blog/odoo-sms-otp-authentication/

You need the Odoo SMS Notification Module with a SMS gateway configured with it.

You also need the Website OTP Authentication Module.

Both of these dependencies should be installed first for Odoo SMS OTP Authentication module to work properly.

Features:

  • Functionality to enable OTP during sign-up
  • Functionality to enable OTP during sign-in
  • Flexibility to login via OTP or password.
  • The OTP validation information along with their success status will be maintained in the backend.
  • Check existing user during sign-up
  • An OTP is sent to a registered mobile no and the user will only be able to register/login on entering the same OTP.
  • Admin can set expiry time and SMS template of OTP.
  • Admin can set OTP type (password or text).
  • Cross browser compatible.
